| ID | 기능 | 설명 | 우선순위 |
|----|------|------|---------|
| T-01 | 투두 추가 | 제목, 우선순위(low/medium/high), 마감일 입력 | High |
| T-02 | 완료 토글 | 체크박스로 완료 상태 전환 | High |
