# Task Log — [TASK-007] 공통 UI 컴팩트 테마

**날짜:** 2026-06-11  
**담당:** 조윤주  
**상태:** `완료`

## 작업 목표

> 스케줄 관리 앱에 맞게 전체 폭과 헤더를 줄이고, 라임 포인트 컬러와 다크 모드 및 압축된 음악 제어 UI를 적용한다.

## 구현 내역

### 변경된 파일

| 파일 | 변경 유형 | 설명 |
|------|----------|------|
| `src/App.tsx` | 수정 | 소형 헤더, Spotify 팝오버, 투두·캘린더 중심 압축 레이아웃 |

| `src/components/player/*` | 수정 | 음악 제어 포인트 컬러 및 하단바 상시 영역 정리 |
| `electron/main.cjs`, `electron/preload.cjs` | 수정 | 기본 창 너비 축소와 캘린더 접힘 시 창 크기 연동 |
| `docs/report/specs/feature-spec.md` | 수정 | U-01~U-05 공통 UI 명세 추가 |

### 주요 결정 사항

- 기본 Electron 창 너비를 1080px로 줄이고 최소 너비는 900px로 설정한다.
- 음악 전용 우측 열을 제거하고 헤더의 작은 `음악` 버튼으로 기존 PlayerView를 팝오버로 제공한다.
- 캘린더를 접으면 본문에는 투두리스트와 52px 펼치기 열만 남긴다.
- 다크 모드는 시스템 설정과 분리해 사용자가 선택한 값을 Zustand persist로 저장한다.
