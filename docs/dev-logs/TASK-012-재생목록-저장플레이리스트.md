## 주요 동작

- 현재 재생 목록을 이름과 함께 저장해 여러 플레이리스트를 생성한다.
- 저장 플레이리스트를 불러오면 현재 재생 목록을 해당 트랙들로 교체한다.
- 현재 재생 목록, 저장 플레이리스트 목록, 저장된 곡 순서를 드래그로 변경한다.
- 저장 플레이리스트의 이름을 변경하거나 삭제할 수 있다.
- 검색 결과의 재생 버튼은 곡을 현재 재생 목록에 추가하고 바로 재생한다.
- 다음/이전 버튼은 곡이 두 개 이상이면 즉시 인접 곡을 재생한다.
- 현재 재생 목록에 한 곡만 있으면 다음/이전 버튼은 현재 곡을 처음부터 재생한다.
- 검색은 300ms 자동 검색만 사용하며 별도 검색 버튼은 표시하지 않는다.

## 데이터 저장

- `playlist`: 현재 재생 목록
- `savedPlaylists`: 추후 다시 불러올 수 있는 저장 플레이리스트 목록
- 두 데이터는 Zustand persist를 통해 localStorage에 저장한다.
