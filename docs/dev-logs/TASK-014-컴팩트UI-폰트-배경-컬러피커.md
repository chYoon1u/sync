## 주요 결정 사항

- 컴팩트 콘텐츠는 390px 너비를 유지한다.
- 투두 영역은 390×390, 하단 재생바는 72px로 구성해 전체 콘텐츠 높이를 462px로 고정한다.
- 컴팩트 모드에서는 상단 앱 헤더를 렌더링하지 않는다.
- 컴팩트 Electron 창은 크기 조절을 잠그고 해제 시 원래 창 크기와 조절 기능을 복원한다.
- 화면 폭에 따라 루트 폰트와 제목 크기가 변하던 미디어쿼리를 제거한다.
- 최외곽 배경은 `#f8f8f8`, 투두·캘린더·헤더·플레이어는 흰색을 유지한다.
- 투두와 캘린더 컨테이너의 그림자를 제거하고 border만 유지한다.
- 컬러 피커에서 선택한 한 색을 기준으로 hover, soft, border, text 컬러를 `color-mix()`로 계산한다.
