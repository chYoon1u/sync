### 주요 결정 사항

- 기본 필터는 오늘 투두이며 새 항목의 날짜도 오늘로 자동 지정한다.
- 사용자 드래그 순서를 유지하기 위해 우선순위 자동 정렬을 제거한다.
- 축소 모드는 앱 전체를 작은 오늘 투두 메모장 화면으로 전환한다.
- 항상 위 표시는 Electron 축소 모드에서만 제공하고 일반 브라우저에서는 비활성화한다.
