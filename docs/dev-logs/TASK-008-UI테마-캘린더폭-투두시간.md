# Task Log — [TASK-008] UI 테마·캘린더 폭·투두 시간

**날짜:** 2026-06-02  
**담당:** 조윤주  
**상태:** `완료`

## 작업 목표

> 캘린더 접힘에 맞춰 창 너비를 자동 조절하면서 투두 패널 폭을 고정하고, 포인트 컬러 설정과 흰색 배경 및 투두 완료 시간을 추가한다.

### 주요 결정 사항

- 캘린더 접힘 여부와 관계없이 투두 패널은 항상 300px을 유지한다.
- 포인트 컬러는 CSS 변수로 적용해 페이지 재구성 없이 즉시 변경한다.
- 밝은 모드의 앱·패널·헤더 배경은 흰색으로 통일한다.
- 완료 시간은 선택값이며 날짜와 독립적으로 비워둘 수 있다.
